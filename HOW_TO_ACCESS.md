## 🎬 الخطوات بالترتيب

### للوصول السريع للبحث المتقدم:

```
1️⃣  شغل التطبيق:
    $ npm run dev

2️⃣  افتح المتصفح:
    http://localhost:5173

3️⃣  ستجد زراً أخضر كبير في منتصف الصفحة:
    [🔍 البحث المتقدم عن المستفيدين]

4️⃣  اضغط عليه!

5️⃣  أدخل رقم هوية (مثل: 900234567)

6️⃣  اضغط "بحث"

7️⃣  استمتع بالتبويبات الثلاث! 🎉
```

## 💡 نصائح سريعة

✅ **لا تحتاج تسجيل دخول** - كل شيء متاح مباشرة!

✅ **الزر الأخضر** هو أسرع طريقة للوصول

✅ **القائمة الإدارية** تعطيك وصول لكل شيء

✅ **البيانات التجريبية** راجع ملف `TEST_DATA_SETUP.md`

---

## 🆘 مشاكل شائعة وحلولها

### المشكلة: لا أجد الزر الأخضر
**الحل:** تأكد أنك في الصفحة الرئيسية، الزر موجود بعد حقل البحث

### المشكلة: لا تظهر نتائج البحث
**الحل:** تأكد من:
1. إضافة بيانات تجريبية (راجع TEST_DATA_SETUP.md)
2. اتصال Supabase يعمل
3. رقم الهوية صحيح (9 أرقام)

### المشكلة: القائمة الإدارية لا تفتح
**الحل:** اضغط على "الدخول الإداري" في أعلى يمين الصفحة
